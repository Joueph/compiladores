# V0.1 - Compilador de análise lexical

Nota: As instruções levam em consideração a utilização do GCC.

## Como utilizar

- Clone o repositório
- Rode o comando
```
gcc analisador-lexical.c -c -o analisador-lexical
```
```
.\analisador-lexical
```

Se aparecer opção de onde abrir o output escolhe notas
